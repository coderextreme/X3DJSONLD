The Annotated Sojourner Model And Simulation

What you should see:
You should see this HTML frame and a frame above with a VRML model of the Sojourner martian rover. You should also see two "sliders" on either side of the Sojourner. If you do not see these sliders, you may have to give the model a "nudge" with the navigation controls in your VRML browser. This nudge should get the sliders to appear.

What can you do?:
1) Try clicking on various components of Sojourner. This frame will update giving information about each of Sojourner's sub-systems.
2) Try moving the two sliders on either side of Sojourner.
Left Slider: The slider on the left controls Sojourner's suspension. Moving this slider allows you to see how the rocker/bogey suspension system allows Sojourner to navigate over very rough terrain.
Right Slider: The slider on the right allows you to steer Sojourner's four control wheels.
